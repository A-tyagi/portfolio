UX Work

Case Study: Uconnect Navigation System

Redesign of the Uconnect nav system in order to create a more accessible interface for drivers who often need to divide their attention between their infotainment systems and the road.

MusicHub Mobile & Web App Design

MusicHub is a social media application centered around listening to music together. On this platform, users will be able to discover, discuss and share music created by the community as well as major artists around the world. Project documentation includes all steps of the design process from ideation to functional prototypes and usability testing.

Instructor Assignment Sorter

This project was done for our sponsor Dreams For Schools, a non profit org running programs for teaching youth about STEM. Our goal was to create an app to allow DFS to easily manage their programs and instructors and also create optimal matches based on location/availability.

Wireframes & Axure Demo

In this project we are presented several sample personas that depict the goals and frustrations of UCI students in the B.S Informatics program. Based on these personas, my task was to generate wireframes to help redesign the undergraduate degree page.




Dev Projects

TaskNote App

TaskNote is an easy to use web app that offers a simple and intuitive way for users to create and organize their daily notes and tasks. Developed using Angular framework.

Preference Learning Weather App

This JavaScript, CSS and HTML based weather app makes use of OpenWeather REST API to fetch the weather forecast for a given city. Users can interact with the app through the thumbs up and thumbs down button to rate how they feel about the current weather.

FabFlix: Movie Rental Ecommerce System

End to end rental system that allows for checkout via PayPal. The design consists of a React frontend, API Gateway and multiple microservices that access a MySql db for identity management, shopping cart and payment processing etc.

Web Crawler & Search Engine

A two part project that involved implementing the core function of a web crawler and creating a search engine from the ground up that is capable of handling thousands of documents and Web pages.

Mobile Sleep Tracker App

Created a simple mobile application for tracking general sleepiness and logging sleep sessions. Developed using Angular and Ionic frameworks to accommodate cross platform conventions across iOS and Android.
