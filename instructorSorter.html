Instructor Assignment Sorter

An instructor management system created for our client Dreams For Schools.



Client & Project Overview

Dreams for Schools is a non-profit organization that oversees programs and initiatives that contribute to educating youth about STEM. As the organization works to scale up their operations, the work-hours needed and complexity to assign each instructor to the appropriate program based on their preferences and location has also increased. The Instructor Assignment Sorter aims to assist program directors in managing and assigning their instructors to schools across California. The goal was to reduce the number of hours needed for instructor assignments so that Dreams For Schools could refocus efforts elsewhere.

Feature Set

  • Ability to import instructor data through bulk CSV upload
  • Crud for managing instructors, schools, programs and classes
  • Drag and drop for manually assigning instructors to classes
  • Auto-assign functionality to assign instructors based on optimal distance and preference

Tech Stack

  • React for our front end since it allows for easy modularity and reusable components.
  • Express and MySql database for our backend.
  • Used SQL as we needed to define constraints and relationships between students, instructors and programs.
  • All authentication was handled through firebase. Users must be signed in and front end requests send a token in the header which are being validated in the back end before providing access.
